## The Story

> *"14 months keeping Microsoft's Azure SQL databases alive in production. Now I build systems and solve hard problems."*

**B.Tech CSE (2022).** Spent 14 months at **HCL Technologies** as an **Azure SQL DB Technical Support Engineer** — **Microsoft** as the direct client. Not a sandbox. Real production. Real deadlocks, real escalations, real stakes.

I know Azure SQL from the inside: query performance, backup strategies, geo-replication, private endpoints, Entra ID auth, RBAC — handled in production under SLA pressure.

Now I focus on what separates good engineers from great ones: **algorithms, mathematics, and systems thinking**. Competitive programming, Python, and the kind of CS depth that gets you through FAANG screens.

Target: **Big Tech. GATE. No shortcuts.**
