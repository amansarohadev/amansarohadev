## 🎯 The Origin Story

> *"I spent 14 months keeping Microsoft's Azure SQL databases alive in production at HCL Technologies. Now I'm building the data pipelines and warehouses behind them."*

I'm a **B.Tech graduate (2022)** with a background that runs through **enterprise Microsoft support → deliberate cloud engineering pivot → live 2026 portfolio sprint**.

While most engineers learn in sandboxes, I was diagnosing **critical production SQL deadlocks, performance bottlenecks, and high-priority escalations** as an **Azure SQL DB Technical Support Engineer at HCL Technologies** — with **Microsoft as the direct client**.

In January 2024, I made a calculated decision: go deep into **Azure Data Engineering**. I call the 26-month period that followed my *"Advanced Cloud Residency"*.

Today I'm applying for **Data Analyst** and **Azure Data Specialist** roles simultaneously — and actively committing to GitHub every week.
